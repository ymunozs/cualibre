    ACERCA DE CUA-LIBRE

    Cua-libre está pensando para que las limitaciones monetarias nunca más sean un obstáculo a la hora de introducirse al bello y fino arte de la escucha y la interpretación cualitativa. Puedes compartir, distribuir y socializar este software con quien quieras y cuantas veces desees.

    El código base de la aplicación está en el github del proyecto, por lo que si tienes una idea de mejora, comentario o una propuesta de expansión, no dudes en comunicarme a ymunozsalinas@gmail.com.

    Si este proyecto te gustó, te invito a realizar una donación en mi perfil de Ko-fi. Todo el dinero recaudado será puesto a disposición de fundaciones que apoyen los procesos vitales de personas trans en Chile.

    Cua-libre está diseñado para que todas las decisiones mantengan autonomía y agencia humana.

    Con amor para Alexandra <3
